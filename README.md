# ~~Fusion.WeiboLite~~
`Abolished`

## 关于
应用基于[微博HTML5版](https://m.weibo.cn)封装架。构

## 使用
安装`release.apk`。

## 再编译
安装[`Fusion App`](https://www.coolapk.com/apk/cn.coldsong.fusionapp)复制项目至`/Fusion App/Projects/`，重启Fusion App。
